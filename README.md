# Hotel Booking Power BI Dashboard

## Overview
This project presents an interactive Power BI dashboard developed for hotel booking analysis. The dashboard provides meaningful insights into booking trends, cancellations, customer behavior, occupancy, and revenue performance to support data-driven decision-making.

---

## Features
- Booking trend analysis
- Hotel cancellation insights
- Revenue and occupancy analysis
- Customer segmentation
- Country-wise booking distribution
- Monthly and yearly performance tracking
- Interactive filters and slicers

---

## Technologies Used
- Power BI
- Microsoft Excel / CSV
- Data Visualization
- Data Analysis

---

## Dashboard Insights
The dashboard helps in:
- Identifying peak booking periods
- Analyzing cancellation rates
- Understanding customer preferences
- Monitoring hotel performance
- Improving business decision-making
